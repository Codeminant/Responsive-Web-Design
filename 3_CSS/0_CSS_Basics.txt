Syntax

    p {
        color: green;
        font-size: 14px;
    }

3 Ways To Add CSS

 1.Inline Style

    <p style="color:green; font-size:14px"> Paragraph </p>

 2.Internal Style Sheet 

    <head>
        <style>
        p {
            color: green;
            font-size: 14px;
        }
        </style>
    </head>

 3.External Style Sheet 
    External CSS are stored in CSS Files
    Link Tag is used to add the External Style Sheet to the document. 
    Syntax :

    <head>
        <link rel="stylesheet" type="text/css" href="style.css">
    </head>
